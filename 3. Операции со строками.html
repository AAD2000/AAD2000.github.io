3. Операции со строками

Инициализация строк

var a: Character = "a"
var hello: String = "hello"
var str1: String = ""
var str2: String = String()

Управляющие последовательности

Специальные символы

  • \n: перевод на новую строку
  • \t: табуляция
  • \": кавычка
  • \\: обратный слеш

Многострочный текст

let text = "ООО \"Рога и копыта 777\" \nГен. директор: Иванов"
let text1 = """
ООО "Рога и копыта 777"
Ген. директор: Иванов
"""
//text индентичен text1

Преобразование в строку

var number: Int = 22
var str: String = String(number) // "22"

Конкатенация строк

var str: String = "" // ""
str += "hello" // "hello"
str = str + " world" // "hello word"

Интерполяция строк

var age: Int = 22
var str: String = "Age: \(age)" // "Age: 22"
var weight: Double = 70.58
str = "Age: \(age) and weight: \(weight)" // "Age: 22 and weight: 70.58"
let x1 = 5
let x2 = 6
let str1 = "x1 + x2 = \(x1 + x2)"
print(str1) // x1 + x2 = 11
